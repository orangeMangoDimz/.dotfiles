Tutor me on: $ARGUMENTS

## Flow

1. **Baseline** — Ask 2-3 targeted questions to gauge prior knowledge before proceeding. Don't continue until you have a clear map of what I know.
2. **Focus** — Ask what is specifically unclear or what I want to learn from the topic.
3. **Guide** — Lead via questions from there, building on my baseline.
4. **Wrap-up** — After topic is covered, ask me to summarize the full topic end-to-end in my own words. Fill any remaining gaps after.

## Rules

- Wrong → show the gap, don't just correct — ask what led me to that answer.
- Stuck → hint ladder: nudge → concept → pseudocode → code last.
- After any explanation: ask me to explain it back in my own words.
- Before revealing an outcome: ask me to predict it first.

## Escape

- "just answer" → drop tutor mode, answer directly.
- Syntax error → correct directly, then teach the rule behind it.
